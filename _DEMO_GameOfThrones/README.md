# 🐉 ДЕМО — Игра Престолов

> *Демонстрация системы на персонажах «Игры Престолов»*

## Что здесь есть

- **25 персонажей** из всех главных домов с полными карточками
- **Семейные связи** — браки, дети, родство
- **Интерактивная визуализация** — открой `FamilyTree.html` в браузере

## 🌐 Визуализация

**Открой файл `FamilyTree.html` в любом браузере** — это standalone файл, не требует сервера.

### Возможности:
- 🔀 **Два режима** — Генеалогическое дерево по поколениям / Граф связей (с физикой)
- 🏠 **Фильтр по домам** — показывай/скрывай любые дома одним кликом
- 👆 **Клик на персонажа** — боковая панель с биографией, титулами, датами
- 🖱️ **Навигация** — прокрутка для масштаба, перетаскивание для панорамы
- 📅 **Таймлайн** — шкала жизней всех персонажей с историческими событиями
- 🎨 **Цвет по домам** — Старки синие, Ланнистеры золотые, Таргариены красные...

## Dataview — полезные запросы

**Все погибшие персонажи:**
```dataview
TABLE дата_смерти AS "Год гибели", место_смерти AS "Место"
FROM "_DEMO_GameOfThrones/Persons"
WHERE дата_смерти != null
SORT дата_смерти ASC
```

**По домам:**
```dataview
TABLE WITHOUT ID дом AS "Дом", count(rows) AS "Кол-во"
FROM "_DEMO_GameOfThrones/Persons"
GROUP BY дом
```
