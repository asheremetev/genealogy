## 👥 Связанные персоны

### Родились здесь

```dataview
TABLE WITHOUT ID
  file.link AS "Персона",
  дата_рождения AS "Дата"
FROM "Persons"
WHERE место_рождения = this.file.link
SORT дата_рождения ASC
```

### Умерли здесь

```dataview
TABLE WITHOUT ID
  file.link AS "Персона",
  дата_смерти AS "Дата"
FROM "Persons"
WHERE место_смерти = this.file.link
SORT дата_смерти ASC
```

### Жили здесь

```dataview
LIST
FROM "Persons"
WHERE contains(места_жизни, this.file.link)
```
