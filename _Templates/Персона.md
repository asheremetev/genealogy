## 👨‍👩‍👧 Семья

**Отец:** `= this.отец`
**Мать:** `= this.мать`

**Семьи/браки:**

```dataview
LIST without ID "[[" + file.name + "]]" + " (" + дата_брака + ")"
FROM "Families"
WHERE contains(муж, this.file.name) OR contains(жена, this.file.name)
```

**Дети:**

```dataview
LIST without ID "[[" + file.name + "]]" + " (" + дата_рождения + ")"
FROM "Persons"
WHERE отец = "[[" + this.file.name + "]]" OR мать = "[[" + this.file.name + "]]"
SORT дата_рождения ASC
```
