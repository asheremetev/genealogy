## 👥 Связанные персоны

**Родились здесь:**
```dataview
LIST without ID "[[" + file.name + "]]" + " (" + дата_рождения + ")"
FROM "Persons"
WHERE место_рождения = "[[" + this.file.name + "]]"
SORT дата_рождения ASC
```

**Умерли здесь:**
