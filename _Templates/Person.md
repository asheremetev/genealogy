## 👨‍👩‍👧 Семья

**Отец:** `= this.отец`
**Мать:** `= this.мать`

### Браки / семьи

```dataview
TABLE WITHOUT ID
  file.link AS "Семья",
  дата_брака AS "Дата брака",
  статус_брака AS "Статус"
FROM "Families"
WHERE муж = this.file.link OR жена = this.file.link
SORT дата_брака ASC
```

### Дети

```dataview
TABLE WITHOUT ID
  file.link AS "Ребёнок",
  дата_рождения AS "Рождение",
  место_рождения AS "Место"
FROM "Persons"
WHERE отец = this.file.link OR мать = this.file.link
SORT дата_рождения ASC
```

### Братья и сёстры

```dataview
TABLE WITHOUT ID
  file.link AS "Имя",
  дата_рождения AS "Рождение"
FROM "Persons"
WHERE (отец = this.отец OR мать = this.мать)
  AND file.name != this.file.name
  AND (отец OR мать)
SORT дата_рождения ASC
```

## 📄 Источники

```dataview
TABLE WITHOUT ID
  file.link AS "Источник",
  категория AS "Тип",
  год_документа AS "Год"
FROM "Sources"
WHERE contains(персоны, this.file.link)
SORT год_документа ASC
```
