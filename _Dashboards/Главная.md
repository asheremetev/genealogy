# 🌳 Главная панель — Генеалогический архив

## 📊 Статистика

```dataview
TABLE WITHOUT ID
  length(filter(rows, (r) => r.тип = "персона")) AS "👤 Персон",
  length(filter(rows, (r) => r.тип = "семья")) AS "👨‍👩‍👧 Семей",
  length(filter(rows, (r) => r.тип = "источник")) AS "📄 Источников",
  length(filter(rows, (r) => r.тип = "место")) AS "📍 Мест"
FLATTEN file AS rows
WHERE тип != null
GROUP BY true
```

---

## ⚠️ Требуют внимания

**Без источников:**
```dataview
LIST without ID "[[" + file.name + "]]"
FROM "Persons"
WHERE достоверность = "предположение" OR достоверность = "низкая"
LIMIT 10
```

**Неизвестные даты рождения:**
```dataview
LIST without ID "[[" + file.name + "]]"
FROM "Persons"
WHERE !дата_рождения
LIMIT 15
```

---

## 👤 Все персоны

```dataview
TABLE WITHOUT ID
  "[[" + file.name + "]]" AS "Имя",
  дата_рождения AS "Рождение",
  место_рождения AS "Место рождения",
  дата_смерти AS "Смерть",
  достоверность AS "Достоверность"
FROM "Persons"
SORT дата_рождения ASC
```

---

## 📄 Последние добавленные источники

```dataview
TABLE WITHOUT ID
  "[[" + file.name + "]]" AS "Источник",
  категория AS "Тип",
  год_документа AS "Год",
  оцифровано AS "Оцифрован"
FROM "Sources"
SORT создано DESC
LIMIT 10
```

---

## 🗺️ Географический охват

```dataview
LIST without ID "[[" + file.name + "]]"
FROM "Places"
SORT file.name ASC
```
