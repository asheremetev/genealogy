# 🌳 Генеалогическое исследование

## 📊 Статистика базы

```dataview
TABLE WITHOUT ID статус as "Статус", length(rows) as "Количество"
FROM "People"
GROUP BY статус
```

## 👥 Все люди по поколениям

```dataview
TABLE имя + " " + отчество as "Имя", фамилия as "Фамилия", дата_рождения as "Рождение", место_рождения as "Место", статус as "Статус"
FROM "People"
SORT поколение ASC, фамилия ASC
```

## 🔴 Требуют проверки

```dataview
TABLE имя + " " + фамилия as "Человек", достоверность as "Достоверность", статус as "Статус"
FROM "People"
WHERE статус = "требует проверки" OR достоверность = "низкая" OR достоверность = "легенда"
```

## 📄 Источники без сканов

```dataview
TABLE название as "Источник", тип_источника as "Тип", архив as "Архив"
FROM "Sources"
WHERE скан_есть = false
SORT тип_источника ASC
```

## 🗺️ Все места

```dataview
TABLE тип_места as "Тип", регион as "Регион", современное_название as "Совр. название"
FROM "Places"
SORT название ASC
```

## 📅 Последние изменения

```dataview
TABLE file.mtime as "Изменено"
FROM "People" OR "Families" OR "Sources"
SORT file.mtime DESC
LIMIT 10
```
