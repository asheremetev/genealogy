# 📋 Прогресс исследования

## 🎯 Охват по поколениям

```dataview
TABLE WITHOUT ID
  поколение as "Поколение",
  length(rows) as "Найдено людей",
  length(filter(rows, (r) => r.статус = "подтверждён")) as "Подтверждено",
  length(filter(rows, (r) => r.достоверность = "высокая")) as "Высокая достоверность"
FROM "People"
GROUP BY поколение
SORT поколение ASC
```

## 🗂️ Истории для проверки

```dataview
TABLE заголовок as "История", рассказчик as "Рассказчик", дата_записи as "Записано"
FROM "Stories"
WHERE достоверность = "низкая" OR достоверность = "средняя"
```

## 🏛️ Архивы для запроса (нет скана и нет онлайн)

```dataview
TABLE архив as "Архив", фонд as "Фонд", название as "Документ"
FROM "Sources"
WHERE скан_есть = false AND онлайн_ссылка = ""
```
