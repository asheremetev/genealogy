## 👥 Связанные персоны

```dataview
TABLE WITHOUT ID
  file.link AS "Персона",
  дата_рождения AS "Дата"
FROM "persons"
WHERE место_рождения = this.file.link
SORT дата_рождения ASC
```

### Умерли здесь

```dataview

SORT дата_смерти ASC
```

### Венчались здесь

```dataview
TABLE WITHOUT ID
  file.link AS "Семья",
```

### События здесь

```dataview
TABLE WITHOUT ID
  вид_события AS "Тип",
  дата AS "Дата",

SORT дата ASC
```
