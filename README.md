# 🌳 Генеалогический Vault

## Структура папок

| Папка         | Содержимое                        |
| ------------- | --------------------------------- |
| `People/`     | Заметка на каждого человека       |
| `Families/`   | Семейная ячейка (пара + дети)     |
| `Places/`     | Места — города, сёла, регионы     |
| `Sources/`    | Архивные источники, книги, сайты  |
| `Documents/`  | Описания сканов документов и фото |
| `Stories/`    | Истории, воспоминания, биографии  |
| `Timeline/`   | Ключевые события по годам         |
| `_templates/` | Шаблоны (не трогать)              |
| `_dashboard/` | Сводные страницы и отчёты         |

## Соглашения об именовании

- **Люди:** `Фамилия Имя Отчество (год рождения)` → `Иванов Пётр Сергеевич (1887)`
- **Семьи:** `Фамилия_мужа - Девичья_фамилия_жены (год брака)` → `Иванов - Петрова (1912)`
- **Места:** `Название, Регион` → `Кострома, Костромская губерния`
- **Источники:** `[Тип] Название` → `[ЗАГС] Метрическая книга Костромы 1887`

## Быстрый старт

1. Установи плагины (список ниже)
2. Включи Templater и укажи папку `_templates`
3. Создавай людей через шаблон Person (Ctrl+P → Templater)
4. Связывай людей через [[Ссылки]]

## Рекомендуемые плагины

### Обязательные

- **Dataview** — запросы к базе заметок (как SQL)
- **Templater** — умные шаблоны с переменными
- **MetaEdit** — редактирование YAML frontmatter
- **Obsidian Git** — автосинхронизация с GitHub

### Очень полезные

- **Kanban** — отслеживание прогресса исследования
- **Leaflet** — интерактивные карты прямо в заметках
- **Excalidraw** — рисование деревьев вручную
- **Timeline** — визуальная лента времени
- **Tag Wrangler** — управление тегами

### Дополнительно

- **Attachment Management** — автоорганизация фото и файлов
- **Natural Language Dates** — вводить даты текстом
