---

## Записи

```dataview
TABLE WITHOUT ID
  file.link AS "Запись",
  дата AS "Дата",
  цель AS "Цель",
  результат AS "Результат"
FROM "reports"
WHERE тип = "запись_журнала"
SORT дата DESC
```
