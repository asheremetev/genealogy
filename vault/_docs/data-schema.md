## 📏 Имена файлов

| Тип         | Формат                                                       | Пример                                                    |
| ----------- | ------------------------------------------------------------ | --------------------------------------------------------- |
| Персона (М) | `Фамилия Имя Отчество (год)`                                 | `Иванов Пётр Сергеевич (1887).md`                         |
| Персона (Ж) | `Фамилия_при_рождении Имя Отчество (год)`                    | `Петрова Анна Ивановна (1890).md`                         |
| Без фамилии | `Имя Отчество (год)` или `Имя Отчество.md` (год неизвестен)  | `Иван Андреевич (~1735).md`                               |
| Семья       | `Полное_имя_мужа - Полное_имя_жены_при_рождении (год_брака)` | `Иванов Пётр Сергеевич - Петрова Анна Ивановна (1912).md` |
| Место       | Префикс + название, уточнение в скобках при одноимённых      | см. таблицу ниже                                          |

**Общие правила:**

- **Для женщин** — всегда фамилия при рождении (и в имени файла, и в `[[ссылках]]`). Стабильно при любых браках.
- **Разделитель супругов в имени семьи** — строго `-` (пробел, дефис, пробел).
- **Год приблизительный** — `(~YYYY)`, без пробела после `~`.
- **Год неизвестен** — опусти скобки (`Иван Андреевич.md`).
- **Никаких трейлинг/дублирующих пробелов** — ловится валидацией в [[data-quality]].

Одноимённые места уточняются в скобках: `с. Никольское (Костромская губ).md`.

## 📋 Общие enum'ы и форматы

### Форматы дат

| Точность       | Формат          | Пример       |
| -------------- | --------------- | ------------ |
| Полная         | `YYYY-MM-DD`    | `1887-03-15` |
| Месяц          | `YYYY-MM`       | `1887-03`    |
| Год            | `YYYY`          | `1887`       |
| Приблизительно | `~YYYY`         | `~1887`      |
| Неизвестно     | оставить пустым |              |

### Ссылки

- Всегда `[[Имя файла]]` без пути к папке.
- Имя в ссылке = точное имя файла (с годом в скобках).
- Для списков: `["[[Файл 1]]", "[[Файл 2]]"]`.
- Для женщин — всегда фамилия **при рождении**.

### Перечислимые значения (enum)

Единый источник правды. Валидатор в [[data-quality]] проверяет эти значения для соответствующих полей.

| Поле                   | Допустимые значения                                                                                                                                                          |
| ---------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `пол`                  | `М` · `Ж` · `неизвестно`                                                                                                                                                     |
| `жив`                  | `да` · `нет` · `неизвестно`                                                                                                                                                  |
| `достоверность`        | `высокая` · `средняя` · `низкая` · `предположение`                                                                                                                           |
| `статус_исследования`  | `полностью` · `частично` · `начато` · `не_начато`                                                                                                                            |
| `приоритет`            | `высокий` · `средний` · `низкий`                                                                                                                                             |
| `статус_брака`         | `в_браке` · `вдовство` · `развод` · `неизвестно`                                                                                                                             |
| `тип_места`            | `деревня` · `село` · `город` · `посёлок` · `губерния` · `уезд` · `волость` · `район` · `область` · `страна` · `другое`                                                       |
| `существует`           | `да` · `нет` · `неизвестно`                                                                                                                                                  |
| `вид_события`          | `рождение` · `смерть` · `брак` · `развод` · `миграция` · `мобилизация` · `арест` · `раскулачивание` · `другое`                                                               |
| `категория` (источник) | `метрическая_книга` · `перепись` · `ревизская_сказка` · `архивный_документ` · `фото` · `паспорт` · `письмо` · `справка` · `грамота` · `устный` · `книга` · `сайт` · `другое` |
| `оцифровано`           | `да` · `нет` · `частично`                                                                                                                                                    |
| `состояние` (источник) | `хорошее` · `повреждён` · `частично_читаем` · `копия`                                                                                                                        |
| `источник_истории`     | `устный_рассказ` · `письмо` · `дневник` · `мемуары` · `другое`                                                                                                               |

> **Примечание.** Для `достоверность` у источников значение `предположение` семантически некорректно — источник либо надёжен, либо нет. Валидатор предупредит, но формально значение разрешено, чтобы не иметь нескольких разных enum'ов.

### Служебные поля

| Поле        | Заполнение                                                                          |
| ----------- | ----------------------------------------------------------------------------------- |
| `создано`   | Подставляется автоматически шаблоном Templater (`<% tp.date.now("YYYY-MM-DD") %>`). |
| `обновлено` | То же. Вручную обновлять не нужно (обновляется при редактировании метаданных).      |

## 🔹 Исследование (`тип: исследование`)

Запись в журнале исследования — одна сессия работы с архивом / интернетом / семьёй.
Хранится в `reports/`.

| Поле        | Тип           | Обязательное | Примечание                                    |
| ----------- | ------------- | :----------: | --------------------------------------------- |
| `тип`       | строка        |      ✅      | `исследование`                                |
| `дата`      | дата          |      ✅      | Дата сессии                                   |
| `цель`      | строка        |      ✅      | Что хотел найти / проверить                   |
| `объекты`   | список ссылок |              | Персоны / места, на которые направлена сессия |
| `результат` | строка        |              | Краткое резюме (одно предложение)             |
| `теги`      | список        |              |                                               |
| `создано`   | дата          |      ✅      | Авто                                          |
| `обновлено` | дата          |      ✅      | Авто                                          |

Подробности (что нашёл, где искал, следующий шаг) — в теле заметки по шаблону [[research-entry]].
