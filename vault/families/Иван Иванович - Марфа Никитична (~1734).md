## 👶 Дети

```dataview
TABLE WITHOUT ID
  file.link AS "Ребёнок",
  пол AS "Пол",
  дата_рождения AS "Рождение",
  место_рождения AS "Место",
  дата_смерти AS "Смерть"
FROM "persons"
WHERE
  (отец = this.муж AND мать = this.жена)
  OR (отец = this.муж AND !мать)
  OR (мать = this.жена AND !отец)
SORT дата_рождения ASC
```

## 📄 Связанные источники

TABLE WITHOUT ID
