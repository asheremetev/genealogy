# Шаблоны — Индекс

Шаблоны применяются автоматически при создании файла в соответствующей папке (через Templater).

| Шаблон              | Папка назначения | Когда использовать                                     |
| ------------------- | ---------------- | ------------------------------------------------------ |
| `person.md`         | `persons/`       | Новый человек (предок, родственник, потомок)           |
| `family.md`         | `families/`      | Пара/брак — связывает мужа и жену                      |
| `event.md`          | `events/`        | Историческое событие с участниками (арест, переезд...) |
| `source.md`         | `sources/`       | Архивный документ, фотография, перепись                |
| `place.md`          | `places/`        | Географическое место (деревня, город, губерния)        |
| `story.md`          | `stories/`       | Семейная история, воспоминания, рассказ                |
| `research-entry.md` | `reports/`       | Запись о сессии исследования в архиве или онлайн       |

## Порядок добавления данных

1. Создай персону (`person.md`) → заполни обязательные поля
2. Создай место (`place.md`) если его ещё нет
3. Создай семью (`family.md`) и свяжи с персонами
4. Добавь источники (`source.md`) и прикрепи к персонам
5. При нахождении новых данных веди журнал (`research-entry.md`)
