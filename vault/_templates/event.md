## 👥 Участники

```dataview
TABLE WITHOUT ID
  file.link AS "Персона",
  пол AS "Пол",
  дата_рождения AS "Рождение"
FROM "persons"
WHERE contains(this.участники, file.link)
```

## 👨‍👩‍👧 Связанные семьи
