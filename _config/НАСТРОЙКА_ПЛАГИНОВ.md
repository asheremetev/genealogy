# ⚙️ Настройки Obsidian для этого Vault

## Установка плагинов

Все плагины устанавливаются через: `Settings → Community plugins → Browse`

### Шаг 1 — Включить сторонние плагины

`Settings → Community plugins → отключить "Safe mode"`

### Шаг 2 — Установить и настроить

---

## Dataview

**Установка:** найди "Dataview" в магазине плагинов

**Настройки после установки:**

- Enable JavaScript Queries: ✅ ON
- Enable Inline Queries: ✅ ON
- Date Format: `YYYY-MM-DD`

---

## Templater

**Установка:** найди "Templater" в магазине плагинов

**Настройки:**

- Template folder location: `_Templates`
- Trigger Templater on new file creation: ✅ ON

**Как использовать:**
`Ctrl+P` → введи `Templater` → выбери `Insert Template` → выбери нужный шаблон

---

## Obsidian Git

**Установка:** найди "Obsidian Git" в магазине плагинов

**Предварительно:** создай репозиторий на GitHub и склонируй его локально, затем открой эту папку как Vault в Obsidian.

**Настройки:**

- Vault backup interval (minutes): `30`
- Auto pull interval (minutes): `10`
- Commit message: `genealogy: vault backup {{date}}`
- Pull updates on startup: ✅ ON

---

## Leaflet (карты)

**Установка:** найди "Obsidian Leaflet" в магазине плагинов

**Использование в заметке:**

````
```leaflet
id: unique-map-id
height: 400px
coordinates: [54.1961, 37.6182]
zoom: 12
marker: default, 54.1961, 37.6182, "Место события"
```
````

---

## DB Folder

**Установка:** найди "DB Folder" в магазине плагинов

Позволяет открыть папку `Persons` как таблицу и редактировать поля прямо в ней — очень удобно для массового ввода данных.

---

## Breadcrumbs (иерархия)

**Установка:** найди "Breadcrumbs" в магазине плагинов

**Настройки:**
Укажи поля для иерархии:

- `отец` → тип: **up** (вверх)
- `мать` → тип: **up** (вверх)

Это позволит строить навигацию вверх по предкам.

---

## Настройки самого Obsidian

### Файлы и ссылки

- Default location for new notes: `Persons`
- Default location for new attachments: `Media`
- Use [[Wikilinks]]: ✅ ON
- Detect all file extensions: ✅ ON

### Основные

- Auto pair brackets: ✅ ON
- Fold heading: ✅ ON

---

## .gitignore для репозитория

Создай файл `.gitignore` в корне vault:

```
.obsidian/workspace.json
.obsidian/workspace-mobile.json
.trash/
.DS_Store
```

Это предотвратит конфликты при синхронизации между устройствами.
