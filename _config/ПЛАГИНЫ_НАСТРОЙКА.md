# ⚙️ Настройка плагинов

## Dataview

Settings → Dataview:

- ✅ Enable JavaScript Queries
- ✅ Enable Inline Queries
- Date Format: `YYYY-MM-DD`

---

## Templater

Settings → Templater:

- Template folder location: `_templates`
- ✅ Trigger Templater on new file creation
- ✅ Enable Folder Templates

Folder Templates (привязка шаблона к папке):
| Папка | Шаблон |
|-------|--------|
| People | `_templates/Person.md` |
| Families | `_templates/Family.md` |
| Places | `_templates/Place.md` |
| Sources | `_templates/Source.md` |
| Documents | `_templates/Document.md` |
| Stories | `_templates/Story.md` |

---

## Obsidian Git

Settings → Obsidian Git:

- Vault backup interval: `10` минут
- ✅ Auto pull on startup
- ✅ Disable push on auto backup (если хочешь контролировать пуш вручную)
- Commit message: `vault backup: {{date}}`

Первичная настройка:

1. Создай репозиторий на GitHub (приватный!)
2. В папке vault запусти: `git init && git remote add origin <url>`
3. Obsidian Git подхватит автоматически

---

## Leaflet (карты)

Вставить в заметку о месте:

```leaflet
id: уникальный-id
lat: 57.7676
long: 40.9269
zoom: 10
marker: default, 57.7676, 40.9269, "Кострома"
```

## Graph View — рекомендуемые настройки

Filters → отключи показ `_templates` и `_dashboard`:

- Exclude files: `_templates, _dashboard`

Groups (цветовая маркировка):

- `path:People` → синий
- `path:Families` → зелёный
- `path:Places` → оранжевый
- `path:Sources` → фиолетовый
