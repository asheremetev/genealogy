## Как демо показывает возможности системы

| Возможность                      | Где видно в демо                                                                 |
| -------------------------------- | -------------------------------------------------------------------------------- |
| **3+ поколения вглубь**          | Поколения -1 → 0 → 1 → 2 (дети → Гарри → родители → деды)                        |
| **Разный уровень информации**    | Гарри — полностью задокументирован; Юфимия — почти ничего не известно            |
| **Связи через ссылки**           | `отец: [[James Potter]]` → Dataview автоматически собирает детей, семьи, братьев |
| **Семья как отдельная сущность** | 4 файла Family связывают мужа, жену и собирают детей через Dataview              |
| **Места с картами**              | Leaflet-карты в Godric's Hollow, The Burrow, Hogwarts                            |
| **Обратные связи**               | Place автоматически показывает «кто родился / умер / жил здесь»                  |
| **Источники → факты**            | Hogwarts Registry → таблица извлечённых фактов по персонам                       |
| **Устные истории**               | «Ночь в Годриковой Впадине» — проверка фактов, статусы                           |
| **События**                      | Битва за Хогвартс — хронология, участники, потери                                |
| **Достоверность**                | `высокая` (Гарри) → `средняя` (Флимонт) → `низкая` (Юфимия)                      |
| **Тупики исследования**          | Линия Эвансов — маглы, нет магических архивов                                    |
| **Задачи как чеклисты**          | `- [ ]` в каждом файле → дашборд собирает все открытые задачи                    |
| **Журнал исследования**          | research entry — что искал, что нашёл, следующий шаг                             |
| **Дашборд**                      | Автоматическая статистика, прогресс по поколениям, проблемные места              |
