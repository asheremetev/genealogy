## 📖 Описание

Финальное сражение Второй магической войны, произошедшее 2 мая 1998 года в замке и на территории Хогвартса.

### Ключевые события для исследуемых персон

| Время  | Событие                                  | Персона                                                |
| ------ | ---------------------------------------- | ------------------------------------------------------ |
| ~00:00 | Прибытие в Хогвартс через портрет Арианы | [[Harry James Potter (1980)]]                          |
| ~01:00 | Эвакуация младших студентов              | [[Ginevra Molly Weasley (1981)]] (вернулась сражаться) |
| ~02:00 | Взрыв стены — гибель                     | [[Fred Weasley (1978)]]                                |
| ~04:00 | Дуэль с Беллатрисой Лестрейндж — победа  | [[Molly Prewett (1949)]]                               |
| ~05:00 | Финальная дуэль с Волдемортом — победа   | [[Harry James Potter (1980)]]                          |

### Потери среди близких

- **Fred Weasley** — погиб от взрыва (Августус Руквуд)
- Remus Lupin — погиб (друг семьи)
- Nymphadora Tonks — погибла (родственница через Блэков)

## 📝 Примечания

Битва стала переломным событием — конец правления Волдеморта, начало восстановления магического общества. Погибли ~50 защитников Хогвартса.

Для семьи Уизли — тяжелейшая утрата (Фред). Для Гарри — завершение миссии, начатой пророчеством.
