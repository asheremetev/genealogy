### Браки / семьи

```dataview
TABLE WITHOUT ID
  file.link AS "Семья",
  дата_брака AS "Дата брака",
  статус_брака AS "Статус"
```

### Дети

```dataview
TABLE WITHOUT ID
  file.link AS "Ребёнок",
  дата_рождения AS "Рождение"
FROM "_demo/HP/persons"
WHERE отец = this.file.link OR мать = this.file.link
SORT дата_рождения ASC
```

### Братья и сёстры

```dataview
TABLE WITHOUT ID
  file.link AS "Имя",
  дата_рождения AS "Рождение"
FROM "_demo/HP/persons"
WHERE (отец = this.отец OR мать = this.мать)
  AND file.name != this.file.name
  AND (отец OR мать)
SORT дата_рождения ASC
```

## 📖 Биография и воспоминания

Младшая из семи детей Артура и Молли Уизли, единственная дочь. Первая девочка, родившаяся в роду Уизли за несколько поколений.

Училась на факультете Гриффиндор (1992–1999). Талантливая ведьма, отличный игрок в квиддич (охотник). Участница Армии Дамблдора и Битвы за Хогвартс.

После Хогвартса играла за команду «Холихедские Гарпии», затем перешла на работу спортивным корреспондентом Daily Prophet.
